Project Athena

Athena is an AI-powered research assistant built to help students, researchers, and scholars work smarter with academic documents. It ingests complex PDF files, converts them into embeddings using FAISS vector stores, and performs advanced semantic search for faster, more relevant information retrieval. By integrating Hugging Face LLMs with a robust RAG (Retrieval-Augmented Generation) pipeline, Athena enables users to ask context-aware questions, generate summaries, and extract key insights in natural language.

The intuitive Gradio interface makes it easy to upload files, query the AI, and get clear answers without switching tools. Designed with scalability in mind, Athena demonstrates my passion for combining LLMs, vector databases, and user-friendly interfaces to solve real research challenges.

My next goal is to deploy Athena on the cloud with multi-user support, making AI-assisted research more accessible for students and institutions worldwide.

AI Driven Navigation for Spinal Robotics

An advanced AI-driven path planning framework for flexible surgical robotic tools in minimally invasive spinal procedures. The system integrates classical A* and Rapidly-Exploring Random Tree (RRT) algorithms for real-time, collision-free trajectory optimization within complex anatomical environments. Preoperative MRI brain images are preprocessed using Python, OpenCV, NumPy, and scikit-image, with deep neural network models developed in TensorFlow and PyTorch for accurate segmentation of spinal structures. Code is authored and version-controlled in Visual Studio Code (VS Code) and GitHub, ensuring modular, reproducible pipelines. The path planning module interfaces with Robot Operating System (ROS) for robotic tool simulation and control, while Docker containers ensure consistent deployment across environments. Research outcomes are being prepared for peer-reviewed publication, contributing to advancements in intelligent robotic-assisted surgery.
